### 📦 Open Source

<a href="https://github.com/Saluja09/knowhook">
  <img src="https://github-readme-stats.vercel.app/api/pin/?username=Saluja09&repo=knowhook&theme=github_dark&hide_border=true&bg_color=0d1117&title_color=c4f048&icon_color=c4f048&text_color=9a9a9e" />
</a>

**[knowhook](https://pypi.org/project/knowhook/)** — Context retention tool for **Claude Code**

- Hooks into Claude Code's compaction lifecycle to snapshot & restore context automatically
- Zero-dependency hook system: `PreCompact`, `SessionStart`, `PostToolUse`
- AST-based dependency mapping across **5 languages** (Python / JS / TS / Go / Rust)
- **154 unit tests** · Pydantic models · Typer CLI + Rich TUI · Published on PyPI

```bash
pip install knowhook
```
